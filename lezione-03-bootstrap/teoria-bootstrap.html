Lezione 3 — Appunti con esempi visivi

1. Come collegare Bootstrap

Bootstrap è una libreria di CSS (e JavaScript) già pronta. Non serve scaricare né installare nulla sul computer: basta collegare la pagina a un CDN, cioè un server esterno che ospita già i file di Bootstrap e li invia al browser quando servono.

Servono due righe di codice:

  • un tag <link> dentro l'<head>, che collega il CSS di Bootstrap (dà lo stile a tutto);
  • un tag <script> messo appena prima di </body>, che collega il JavaScript di Bootstrap (serve solo per i componenti interattivi, come menu a tendina o alert chiudibili).

Esempio: pagina HTML minima con Bootstrap collegato

<!DOCTYPE html>
<html lang="it">
<head>
  <meta charset="UTF-8">
  <title>La mia pagina</title>

  <!-- Tag link: collega il CSS di Bootstrap -->
  <link
    href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css"
    rel="stylesheet"
  >
</head>
<body>

  <div class="container py-5">
    <h1>Ciao Bootstrap!</h1>
    <p>Questa pagina usa Bootstrap senza aver scaricato nulla.</p>
    <button class="btn btn-primary">Cliccami</button>
  </div>

  <!-- Tag script: collega il JS di Bootstrap, va prima di  -->
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>

2. Griglia — container / row / col

Bootstrap divide ogni riga della pagina in 12 colonne invisibili. Con le classi col-N decidi quante di quelle 12 colonne occupa un elemento: col-12 occupa tutta la riga, col-6 ne occupa metà, col-4 un terzo. La somma delle colonne usate in una riga deve fare 12. La struttura è sempre container > row > col.

Esempio: tre colonne uguali (col-4 + col-4 + col-4)

<div class="container">
  <div class="row g-3">
    <div class="col-4">col-4</div>
    <div class="col-4">col-4</div>
    <div class="col-4">col-4</div>
  </div>
</div>
Risultato
col-4 (33%)
col-4 (33%)
col-4 (33%)

Esempio: due colonne asimmetriche (col-8 + col-4)

<div class="row g-3">
  <div class="col-8">contenuto principale</div>
  <div class="col-4">sidebar</div>
</div>
Risultato
col-8 (contenuto principale)
col-4 (sidebar)

Esempio: colonne responsive (col-md-6)

<!-- col-12 su mobile (piena larghezza), col-md-6 da tablet in su (metà) -->
<div class="row g-3">
  <div class="col-12 col-md-6">Prima metà</div>
  <div class="col-12 col-md-6">Seconda metà</div>
</div>
Risultato
Prima metà
Seconda metà
Ricorda: la griglia funziona solo dentro un .container (o .container-fluid).

3. Spacing — mt / mb / ms / me / px / py

Bootstrap gestisce margini e padding con classi brevi, sempre nella stessa forma: {proprietà}{lato}-{valore}.

  • Proprietà: m = margin (fuori dal bordo), p = padding (dentro il bordo)
  • Lato: t = top (sopra), b = bottom (sotto), s = start (sinistra), e = end (destra), x = sinistra+destra, y = sopra+sotto
  • Valore: un numero da 0 a 5, ognuno pari a 0.25rem in più (0, 4px, 8px, 16px, 24px, 48px)

Esempi: mt-3 = margin-top medio, mb-0 = nessun margin sotto, ms-auto = spinge l'elemento a destra, px-4 = padding sinistra e destra, py-5 = padding sopra e sotto molto grande.

Esempio: differenza tra padding piccolo, medio e grande

<div class="px-2 py-1">px-2 py-1 (piccolo)</div>
<div class="mt-3 px-4 py-3">px-4 py-3 (medio)</div>
<div class="mt-3 px-5 py-5">px-5 py-5 (grande)</div>
Risultato
px-2 py-1 (piccolo)
px-4 py-3 (medio)
px-5 py-5 (grande)

Esempio: margini diversi tra due box (mb-0 vs mb-5)

<div class="p-2 mb-0">mb-0 (nessuno spazio sotto)</div>
<div class="p-2 mb-5">mb-5 (molto spazio sotto)</div>
<div class="p-2">box successivo</div>
Risultato
mb-0 (nessuno spazio sotto)
mb-5 (molto spazio sotto)
box successivo

4. Pulsanti

La classe base è btn, obbligatoria su ogni pulsante. Poi si aggiunge un modificatore di colore come btn-primary, oppure una versione "outline" (solo bordo, senza sfondo pieno). Per la dimensione si usa btn-lg (grande) o btn-sm (piccolo).

Esempio: tutti i tipi di pulsante

<button class="btn btn-primary">Primary</button>
<button class="btn btn-secondary">Secondary</button>
<button class="btn btn-success">Success</button>
<button class="btn btn-danger">Danger</button>
<button class="btn btn-warning">Warning</button>

<button class="btn btn-outline-primary">Outline Primary</button>
<button class="btn btn-outline-danger">Outline Danger</button>
<button class="btn btn-outline-success">Outline Success</button>

<button class="btn btn-primary btn-lg">Grande (btn-lg)</button>
<button class="btn btn-primary">Normale</button>
<button class="btn btn-primary btn-sm">Piccolo (btn-sm)</button>
Risultato
Pulsanti solidi
Pulsanti outline (solo bordo)

5. Form

Bootstrap stila automaticamente input, label, select e textarea. La struttura tipica di ogni campo è: un contenitore con mb-3 (spazio sotto), dentro un'etichetta con form-label e il campo con form-control (o form-select per i menu a tendina).

Esempio: form completo

<form>
  <div class="mb-3">
    <label for="nome" class="form-label">Nome</label>
    <input type="text" id="nome" class="form-control" placeholder="Scrivi il tuo nome">
  </div>

  <div class="mb-3">
    <label for="classe" class="form-label">Classe</label>
    <select id="classe" class="form-select">
      <option value="">-- Scegli --</option>
      <option>Prima</option>
      <option>Seconda</option>
      <option>Terza</option>
    </select>
  </div>

  <div class="mb-3">
    <label for="note" class="form-label">Note</label>
    <textarea id="note" class="form-control" rows="3"></textarea>
  </div>

  <button type="submit" class="btn btn-primary">Invia</button>
</form>

6. Tabella

Aggiungendo la classe table a un elemento <table> si ottiene subito uno stile pulito. Si possono combinare più classi: table-striped (righe alternate colorate), table-hover (la riga si evidenzia al passaggio del mouse), table-bordered (bordi su tutte le celle) e table-dark sull'intestazione per un tema scuro. Nelle celle si possono inserire dei badge colorati per mostrare uno stato a colpo d'occhio.

Esempio: tabella con badge di stato

<table class="table table-striped table-bordered table-hover">
  <thead class="table-dark">
    <tr>
      <th>#</th>
      <th>Nome</th>
      <th>Voto</th>
      <th>Stato</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>1</td>
      <td>Alice</td>
      <td>8</td>
      <td><span class="badge bg-success">Promossa</span></td>
    </tr>
    <tr>
      <td>2</td>
      <td>Bruno</td>
      <td>4</td>
      <td><span class="badge bg-danger">Bocciato</span></td>
    </tr>
    <tr>
      <td>3</td>
      <td>Clara</td>
      <td>5</td>
      <td><span class="badge bg-warning text-dark">Sospesa</span></td>
    </tr>
  </tbody>
</table>
Risultato
# Nome Voto Stato
1 Alice 8 Promossa
2 Bruno 4 Bocciato
3 Clara 5 Sospesa

7. Card

Una card è un riquadro con ombra leggera e una struttura fissa: card-header (intestazione, opzionale), card-body (contenuto principale) che di solito contiene un card-title e un card-text, e card-footer (piè di pagina, opzionale). Si usano per mostrare contenuti in modo ordinato, spesso affiancate in una griglia.

Esempio: card con header, titolo, testo e pulsante

<div class="card" style="width: 18rem;">
  <div class="card-header">
    Header della card
  </div>
  <div class="card-body">
    <h5 class="card-title">Titolo della card</h5>
    <p class="card-text">Un breve testo che descrive il contenuto della card.</p>
    <a href="#" class="btn btn-primary">Vai</a>
  </div>
</div>
Risultato
Header della card
Titolo della card

8. Alert

Gli alert sono messaggi colorati per avvisare l'utente. Basta la classe alert più un modificatore di colore come alert-success, alert-danger, alert-warning o alert-info.

Esempio: i quattro alert più usati

<div class="alert alert-success">
  <strong>Success:</strong> operazione completata con successo!
</div>
<div class="alert alert-danger">
  <strong>Danger:</strong> errore, qualcosa è andato storto.
</div>
<div class="alert alert-warning">
  <strong>Warning:</strong> attenzione, controlla prima di continuare.
</div>
<div class="alert alert-info">
  <strong>Info:</strong> hai un nuovo messaggio.
</div>
Risultato
Success: operazione completata con successo!
Danger: errore, qualcosa è andato storto.
Warning: attenzione, controlla prima di continuare.
Info: hai un nuovo messaggio.
