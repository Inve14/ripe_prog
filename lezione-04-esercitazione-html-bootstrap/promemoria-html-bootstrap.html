Promemoria — HTML & Bootstrap

Foglio di riferimento rapido per l'esercitazione. Se non ricordi una classe, cercala qui.

HTML

Tag principali

<h1>...<h6>

Titoli, dal più importante (h1) al meno importante (h6).

<p>

Un paragrafo di testo.

<a>

Un link. L'indirizzo va nell'attributo href.

<img>

Un'immagine. Serve sempre src e alt.

<ul> / <ol> / <li>

Lista puntata, lista numerata, singolo elemento della lista.

<table><thead><tbody><tr><th><td>

Tabella, intestazione, corpo, riga, cella di intestazione, cella normale.

<form>

Contenitore che raggruppa i campi di un modulo.

<input>

Campo di inserimento dati; il tipo cambia con l'attributo type.

<select>

Menu a tendina con opzioni <option>.

<textarea>

Campo di testo multi-riga.

<button>

Un pulsante cliccabile.

<div>

Contenitore generico "a blocco", usato per organizzare la pagina.

<span>

Contenitore generico "in linea", usato dentro al testo.

Attributo id

L'id è un nome unico dato a un elemento: è il gancio per JavaScript (e per il CSS mirato) — serve a dire "questo elemento specifico, e nessun altro".

div vs span

<div> è a blocco: occupa tutta la riga e va a capo prima e dopo. <span> è in linea: sta dentro al testo senza andare a capo.

Tipi di input più usati

type="text"

Testo libero su una riga.

type="number"

Solo numeri, con frecce su/giù.

type="email"

Testo che il browser verifica abbia il formato di un'email.

type="password"

Testo nascosto con dei pallini.

type="date"

Selettore di data con calendario.

type="checkbox"

Casella spuntabile, sì/no.

Bootstrap

Come collegare Bootstrap

Nel <head>:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

Prima di </body> (serve solo se usi componenti interattivi):

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

Griglia

container → row → col. Regola fondamentale: la somma delle colonne in una riga deve fare 12.

<div class="container">
  <div class="row">
    <div class="col-4">...</div>
    <div class="col-8">...</div>
  </div>
</div>

Breakpoint

ClasseSi attiva da
col-*sempre (mobile compreso)
col-sm-*≥ 576px (piccoli tablet)
col-md-*≥ 768px (tablet)
col-lg-*≥ 992px (desktop)

Spacing

Logica: m(argin) o p(adding) + lato + numero da 0 a 5.

mt / mb

margine sopra / sotto (top / bottom)

ms / me

margine a sinistra / destra (start / end)

px / py

padding orizzontale (x) / verticale (y)

mt-3, p-2, mb-5...

il numero 0-5 indica quanto spazio: 0 = niente, 5 = tanto

Pulsanti

btn

Classe base, sempre obbligatoria.

btn-primary / btn-success / btn-danger / btn-warning

Colore pieno: blu, verde, rosso, giallo.

btn-outline-*

Stessa gamma di colori, ma solo bordo (es. btn-outline-primary).

Classi form

form-label

Etichetta di un campo.

form-control

Stile per input, textarea.

form-select

Stile per i menu a tendina (select).

form-check / form-check-input / form-check-label

Contenitore, casella e testo per checkbox/radio.

Classi tabella

table

Classe base per lo stile Bootstrap.

table-striped

Righe alternate chiaro/bianco.

table-hover

Riga evidenziata al passaggio del mouse.

table-bordered

Bordi visibili su tutte le celle.

table-dark

Sfondo scuro (spesso usata su <thead>).

badge

Etichetta colorata piccola (es. badge bg-success).

Classi card

card

Contenitore principale della card.

card-header

Fascia superiore della card.

card-body

Contenuto principale della card.

card-title

Titolo dentro il body.

card-text

Paragrafo di testo dentro il body.

Classi alert

alert

Classe base, sempre obbligatoria.

alert-success

Verde: operazione riuscita.

alert-danger

Rosso: errore.

alert-warning

Giallo: avviso.

alert-info

Azzurro: informazione neutra.

Classi utility utili

d-flex

Rende il contenitore un flexbox.

align-items-center

Centra gli elementi flex sull'asse verticale.

justify-content-center

Centra gli elementi flex sull'asse orizzontale.

min-vh-100

Altezza minima pari a tutto lo schermo.

w-100

Larghezza 100%.

text-center

Testo centrato.

mt-* / mb-*

Scorciatoie di spacing (vedi sezione Spacing).

list-unstyled

Toglie i pallini/numeri da una lista.
