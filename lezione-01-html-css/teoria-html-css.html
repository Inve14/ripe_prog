// Appunti — Lezione 01: HTML & CSS

1. Struttura base di una pagina HTML

<!-- Ogni file HTML deve iniziare con questa dichiarazione.
     Dice al browser: "questo è un documento HTML5". -->
<!DOCTYPE html>

<!-- <html> è l'elemento radice: contiene TUTTO il resto.
     lang="it" dice al browser che la lingua è italiano. -->
<html lang="it">

<!-- <head> = la "testa" del documento.
     Contiene informazioni INVISIBILI nella pagina:
     titolo, charset, link a CSS, meta dati. -->
<head>
  <meta charset="UTF-8">          <!-- codifica caratteri: include lettere accentate -->
  <title>La mia pagina</title>  <!-- testo che appare sulla scheda del browser -->
</head>

<!-- <body> = il "corpo" della pagina.
     Tutto ciò che scrivi qui viene visualizzato dall'utente. -->
<body>
  <!-- contenuto della pagina -->
</body>

</html>

2. Tag principali

<!-- TITOLI: da h1 (più grande/importante) a h6 (più piccolo).
     h1 = titolo principale della pagina, usalo UNA VOLTA SOLA.
     h2, h3... = sotto-titoli e sezioni. -->
<h1>Titolo principale</h1>
<h2>Sotto-titolo</h2>
<h3>Sezione</h3>

<!-- PARAGRAFO: <p> crea un blocco di testo.
     Il browser aggiunge automaticamente spazio sopra e sotto. -->
<p>Questo è un paragrafo di testo normale.</p>

<!-- LINK: <a> crea un collegamento cliccabile.
     href = "dove vai" quando clicchi.
     target="_blank" = apre in una nuova scheda. -->
<a href="https://esempio.it" target="_blank">Clicca qui</a>

<!-- IMMAGINE: <img> è un tag "auto-chiudente" (non ha tag di chiusura).
     src = percorso o URL dell'immagine.
     alt = testo alternativo se l'immagine non carica (e per accessibilità). -->
<img src="foto.jpg" alt="Descrizione immagine">

<!-- DIV: contenitore generico. Non ha significato semantico.
     Serve per raggruppare elementi e applicare stili CSS.
     È come una "scatola invisibile". -->
<div>
  <p>Elementi raggruppati nel div.</p>
</div>

<!-- SPAN: come il div, ma inline (non va a capo).
     Usato per stilizzare una parte di testo. -->
<p>Il voto è <span style="color:red">8</span>.</p>

<!-- LISTE:
     <ul> = lista NON ordinata (punti elenco)
     <ol> = lista ordinata (numeri)
     <li> = singola voce della lista -->
<ul>
  <li>Primo elemento</li>
  <li>Secondo elemento</li>
</ul>

<!-- TESTO IN GRASSETTO E CORSIVO -->
<strong>Testo importante (grassetto)</strong>
<em>Testo enfatizzato (corsivo)</em>

3. Form e Input

I form servono a raccogliere dati dall'utente. In HTML definiscono la STRUTTURA. JavaScript li rende interattivi. 
<!-- FORM: il contenitore di tutti i campi di input.
     action = dove inviare i dati (URL del server).
     method = come inviare: GET (visibile in URL) o POST (nascosto). -->
<form action="/invia" method="post">

  <!-- LABEL: etichetta collegata a un input tramite "for" = "id" dell'input.
       Cliccando sulla label, il cursore va nell'input. -->
  <label for="nome">Nome:</label>

  <!-- INPUT type="text": campo di testo a riga singola.
       name = nome del campo quando il form viene inviato.
       placeholder = testo grigio suggerito prima che l'utente scriva. -->
  <input type="text" id="nome" name="nome" placeholder="Inserisci nome">

  <!-- INPUT type="number": accetta solo numeri. -->
  <input type="number" id="voto" name="voto" min="1" max="10">

  <!-- INPUT type="email": valida automaticamente il formato email. -->
  <input type="email" id="email" name="email">

  <!-- SELECT: menù a tendina.
       <option> = ogni voce del menù. -->
  <select id="materia" name="materia">
    <option value="matematica">Matematica</option>
    <option value="italiano">Italiano</option>
  </select>

  <!-- BUTTON type="submit": invia il form quando cliccato. -->
  <button type="submit">Invia</button>

</form>

4. L'attributo id — il "gancio" per JavaScript

L'id è come un nome unico sulla carta d'identità: OGNI elemento può averne uno solo, e ogni id deve essere unico nell'intera pagina. 
<!-- id = identificatore univoco di un elemento.
     Regola d'oro: non possono esistere due elementi con lo stesso id. -->
<h1 id="titolo-principale">Benvenuto</h1>
<p  id="messaggio">Testo iniziale</p>
<input id="campo-nome" type="text">

<!-- In JavaScript, per "agganciare" un elemento usi:
     document.getElementById("messaggio")
     Questo restituisce l'elemento HTML e ti permette di modificarlo.

     Esempio (anticipazione JavaScript):
     document.getElementById("messaggio").textContent = "Ciao!"
     → cambia il testo del <p id="messaggio"> in "Ciao!"

     Senza id, JavaScript non saprebbe QUALE elemento modificare
     se ne esistono tanti dello stesso tipo. -->

5. Selettori CSS

Un selettore CSS risponde alla domanda: "a QUALI elementi voglio applicare questo stile?" 
/* ── 1. SELETTORE UNIVERSALE ──
   Il simbolo * seleziona TUTTI gli elementi della pagina.
   Usato spesso per il reset dei margini/padding. */
* {
  margin: 0;
  padding: 0;
}

/* ── 2. SELETTORE DI ELEMENTO (tag) ──
   Si usa il nome del tag. Colpisce TUTTI gli elementi di quel tipo. */
p {
  color: #333;       /* tutti i <p> avranno questo colore */
}
h1 {
  font-size: 2rem;
}

/* ── 3. SELETTORE DI CLASSE (.nome-classe) ──
   Colpisce gli elementi che hanno class="nome-classe".
   La stessa classe può essere usata su più elementi. */
.evidenziato {
  background-color: yellow;
}
/* HTML: <p class="evidenziato">Testo</p> */

/* ── 4. SELETTORE DI ID (#nome-id) ──
   Colpisce l'elemento con id="nome-id". UNO SOLO nella pagina.
   Ha priorità più alta rispetto alle classi. */
#titolo-principale {
  color: navy;
  text-align: center;
}
/* HTML: <h1 id="titolo-principale">...</h1> */

/* ── 5. SELETTORE DISCENDENTE ──
   Spazio tra selettori = "elementi A che stanno DENTRO B". */
div p {
  /* solo i <p> che si trovano dentro un <div> */
  font-style: italic;
}

6. Box Model

In CSS ogni elemento è una "scatola" con quattro strati, dall'interno all'esterno: content → padding → border → margin. 
/*
   Immagina una cornice con un quadro:

   ┌─────────────────────────────────┐
   │           MARGIN                │  ← spazio esterno (tra elementi)
   │   ┌─────────────────────┐       │
   │   │       BORDER        │       │  ← bordo visibile
   │   │  ┌───────────────┐  │       │
   │   │  │    PADDING    │  │       │  ← spazio interno (tra bordo e contenuto)
   │   │  │  ┌─────────┐  │  │       │
   │   │  │  │ CONTENT │  │  │       │  ← il testo/immagine vera e propria
   │   │  │  └─────────┘  │  │       │
   │   │  └───────────────┘  │       │
   │   └─────────────────────┘       │
   └─────────────────────────────────┘

   - content: larghezza/altezza del contenuto (width, height)
   - padding: respiro interno (tra il testo e il bordo)
   - border: il bordo della scatola
   - margin: distanza dagli altri elementi vicini
*/

.card {
  width: 300px;          /* larghezza del contenuto */
  height: 200px;         /* altezza del contenuto */
  padding: 20px;         /* 20px di spazio su tutti e 4 i lati interni */
  border: 2px solid black; /* bordo spesso 2px, solido, nero */
  margin: 10px auto;     /* 10px sopra/sotto, auto = centrato orizzontalmente */
}

/* box-sizing: border-box → il padding e border
   NON si aggiungono alle dimensioni dichiarate.
   width: 300px rimane 300px in totale. */
* {
  box-sizing: border-box;
}

7. Pseudo-classi CSS

Le pseudo-classi selezionano elementi in uno STATO specifico (quando ci passi sopra il mouse, quando sono selezionati, ecc.). Si scrivono con i due punti: elemento:stato. 
/* :hover — si attiva quando il mouse passa sopra l'elemento.
   Usato spesso su link e bottoni per dare feedback visivo. */
button:hover {
  background-color: #0056b3;
  cursor: pointer;   /* cambia il cursore in "manina" */
}

a:hover {
  text-decoration: underline;
  color: red;
}

/* :focus — si attiva quando un elemento riceve il "fuoco" del tastiera.
   Tipico degli input: quando ci clicchi dentro (o navighi con Tab).
   Importante per l'accessibilità! --> */
input:focus {
  outline: 2px solid #4dabf7;  /* bordo blu quando sei dentro l'input */
  background-color: #f0f8ff;
}

/* :nth-child(n) — seleziona l'n-esimo figlio.
   Utile per colorare righe alternate in una tabella. */
tr:nth-child(even) {
  background-color: #f2f2f2;   /* righe pari = sfondo grigio chiaro */
}

/* :first-child / :last-child */
li:first-child {
  font-weight: bold;   /* solo il primo <li> in grassetto */
}
